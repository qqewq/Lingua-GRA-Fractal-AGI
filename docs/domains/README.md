# Lingua‑GRA‑X Domains / Доменные языки Lingua‑GRA‑X

This folder describes **domain‑specific fractal languages** of the form **Lingua‑GRA‑X**  
(e.g. *Lingua‑GRA‑Math*, *Lingua‑GRA‑Cinema*), built on top of the core
**Lingua‑GRA‑Fractal‑AGI** architecture.[web:34][web:401]

В этой папке описаны **доменные фрактальные языки** вида **Lingua‑GRA‑X**  
(например, *Lingua‑GRA‑Math*, *Lingua‑GRA‑Cinema*), построенные поверх базовой
архитектуры **Lingua‑GRA‑Fractal‑AGI**.[web:34][web:401]

Each Lingua‑GRA‑X language combines:  
Каждый язык Lingua‑GRA‑X сочетает в себе:

- a shared multi‑level linguistic backbone (symbolic, syntactic, semantic, pragmatic, meta);  
- общий многоуровневый лингвистический «скелет» (символический, синтаксический, семантический, прагматический, мета‑уровни);  
- domain‑specific structures and objectives (e.g. math, cinema);  
- доменные структуры и цели (например, математика, кино);  
- fractal geometry targets for embeddings derived from domain corpora (e.g. correlation dimension \(D_2\)).[web:34][web:401][web:406]  
- фрактальные геометрические цели для эмбеддингов, извлечённые из доменных корпусов (например, корреляционная размерность \(D_2\)).[web:34][web:401][web:406]
